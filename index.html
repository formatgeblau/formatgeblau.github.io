Formatge Blaau! :D





Pàgina web dedicada al gust fabulós del formatge blau



Aquesta pàgina web és una prova que he volgut fer per veure com funciona tota aquesta història de les pàgines web.
No és una web gaire desenvolupada perquè el que buscava fent aquest projecte en concret no era aprofundir en la creació del codi font d'una web, utilitzant, html, css i altres, cosa que també m'interessa.
Enlloc de la creació en si, l'objectiu d'aquesta web era aprendre sobre com penjar-ne a la xarxa.



He après la majoria de la informació sobre com poder realitzar aquest petit projecte de prova a la web Codecademy (tan sols la versió gratuïta) i investigant i provant pel meu compte.



Les eines utlitzades han estat:

  • L'editor de text per defecte d'Ubuntu, "gedit".
  • El terminal de linux i el software incorporat "git".
  • El servei ofert per la web Github on l'única "pega" que té la versió gratuïta és que el que penjis allà serà públic. Fet que per aquest projecte no m'importa i no és per tant cap inconvenient.
  • L'adquisició d'un domini gratuït per un any a la web Freenom amb el TLD (top-level-domain) ".ga", el TLD de Gabon.


I que visquin els formatges blaus!!! :9
